Short Pressure

Agent Workflow

โครงสร้างการทำงานถูกออกแบบให้ตรวจสอบข้อมูลก่อนวิเคราะห์ และแยกข้อมูลจริงออกจากข้อมูลเสริมที่ต้องใช้ API provider

ค้นหาข้อมูล ดึงราคา, volume, SEC, FINRA และ optional feeds
ตรวจสอบข้อมูล เช็กความสด, missing values และ data quality
คำนวณ Indicator สร้าง technical, flow, sentiment และ risk metrics
สังเคราะห์คะแนน รวมเป็น component score และ composite score
วิเคราะห์ 3 มุม พื้นฐาน+กราฟ, จิตวิทยา และเทคนิคอล
พยากรณ์วันถัดไป ประเมิน bias, probability และ expected range
