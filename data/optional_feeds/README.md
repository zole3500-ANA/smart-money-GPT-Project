# Optional paid/third-party feed files

Put one JSON file per ticker here, for example `AAPL.json`.

These files are optional. The open-source agent works without them, but options-flow, dark-pool, exact short-interest, social sentiment, and full 13F issuer-level aggregation usually require licensed data providers.

See `docs/OPTIONAL_FEED_SCHEMA.json` for a complete example.
